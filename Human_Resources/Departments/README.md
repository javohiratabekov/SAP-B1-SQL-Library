# Departments

SAP B1: **Human Resources → Departments**

Organizational units for employee grouping, cost center alignment, and reporting hierarchy.

## Key Tables
- `OUDP` — Departments
- `OHEM` — Employees (linked via `Department` field)

## Naming Prefix
`HR_DEP_` — e.g., `HR_DEP_Headcount_By_Department.sql`
