# Human Resources Queries

## Purpose
SQL queries for employee data, HR analytics, and workforce management.

## Query Types
- Employee master list
- Department and cost center analysis
- Employee status tracking
- Organizational structure
- Headcount reports
- Employee activity tracking

## Example File Names
- `HR_Employee_Master_List.sql`
- `HR_Headcount_By_Department.sql`
- `HR_Active_Employees_Report.sql`
- `HR_Cost_Center_Analysis.sql`
- `HR_Employee_Status_Report.sql`

## SAP B1 Tables
Common tables used:
- `OHEM` - Employees Master Data
- `OUDP` - Departments
- `OCST` - Cost Centers
- `OACT` - Chart of Accounts (for HR costs)
