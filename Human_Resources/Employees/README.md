# Employees

SAP B1: **Human Resources → Employee Master Data**

Employee records including personal data, employment status, and department.

## Key Tables
- `OHEM` — Employee Master
- `OUDP` — Departments (linked via `Department` field)
- `OBPL` — Branches

## Naming Prefix
`HR_EMP_` — e.g., `HR_EMP_Active_Employees.sql`, `HR_EMP_Headcount_Report.sql`
