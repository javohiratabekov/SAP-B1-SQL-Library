# Inventory Reports Queries

## Purpose
Comprehensive inventory reporting, analytics, and KPI tracking.

## Query Types
- Inventory turnover analysis
- ABC analysis
- Stock aging reports
- Inventory accuracy metrics
- Dead stock analysis
- Inventory KPI dashboards

## Example File Names
- `INV_RPT_Inventory_Turnover_Analysis.sql`
- `INV_RPT_ABC_Classification.sql`
- `INV_RPT_Stock_Aging_Report.sql`
- `INV_RPT_Dead_Stock_Analysis.sql`
- `INV_RPT_Inventory_Accuracy_Report.sql`

## SAP B1 Tables
Common tables used:
- `OITM` - Items Master Data
- `OITW` - Item Warehouse Info
- `OINM` - Inventory Transactions
- `INV1` - AR Invoice Rows (for turnover)
