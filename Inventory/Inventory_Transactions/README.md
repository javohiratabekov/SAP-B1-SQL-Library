# Inventory Transactions Queries

## Purpose
SQL queries for inventory movements, transaction history, and stock adjustments.

## Query Types
- Inventory transaction log
- Stock movements by type
- Goods issue/receipt tracking
- Inventory adjustments
- Transaction audit trail
- Movement analysis

## Example File Names
- `INV_TRX_Transaction_History.sql`
- `INV_TRX_Stock_Movements_Daily.sql`
- `INV_TRX_Goods_Issue_Report.sql`
- `INV_TRX_Inventory_Adjustments.sql`
- `INV_TRX_Movement_By_Item.sql`

## SAP B1 Tables
Common tables used:
- `OINM` - Inventory Transactions
- `IGE1` - Goods Issue Rows
- `IGN1` - Goods Receipt Rows
- `OWTR` - Inventory Transfer Header
