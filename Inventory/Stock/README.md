# Stock Queries

## Purpose
SQL queries for stock levels, availability, and inventory status tracking.

## Query Types
- Current stock levels
- Stock availability
- Reorder point analysis
- Safety stock monitoring
- Stock below minimum
- Item availability by warehouse

## Example File Names
- `INV_STK_Current_Stock_Levels.sql`
- `INV_STK_Items_Below_Minimum.sql`
- `INV_STK_Stock_Availability.sql`
- `INV_STK_Reorder_Point_Analysis.sql`
- `INV_STK_Zero_Stock_Items.sql`

## SAP B1 Tables
Common tables used:
- `OITM` - Items Master Data
- `OITW` - Item Warehouse Info
- `OINV` - Stock Taking
- `OWHS` - Warehouses
