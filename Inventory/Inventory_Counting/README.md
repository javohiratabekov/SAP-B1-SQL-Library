# Inventory Counting

SAP B1: **Inventory → Inventory Counting**

Physical stock count and resulting adjustments to reconcile system quantities vs physical quantities.

## Key Tables
- `OITC` — Inventory Counting Header
- `ITC1` — Inventory Counting Lines
- `OIGE` / `OIGN` — Adjustment transactions created post-count

## Naming Prefix
`INV_CNT_` — e.g., `INV_CNT_Count_Variance_Report.sql`
