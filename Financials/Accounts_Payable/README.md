# Accounts Payable Queries

## Purpose
SQL queries for vendor invoices, payments, aging analysis, and payables management.

## Query Types
- AP aging reports
- Vendor payment status
- Outstanding payables
- Payment runs
- Vendor balances
- Due date analysis

## Example File Names
- `FIN_AP_Aging_Report.sql`
- `FIN_AP_Outstanding_Invoices.sql`
- `FIN_AP_Vendor_Balance.sql`
- `FIN_AP_Payment_Due_This_Week.sql`
- `FIN_AP_Payment_History.sql`

## SAP B1 Tables
Common tables used:
- `OPCH` - AP Invoice Header
- `PCH1` - AP Invoice Rows
- `OVPM` - Outgoing Payments
- `VPM2` - Outgoing Payments - Rows
- `OCRD` - Business Partners (Vendors)
