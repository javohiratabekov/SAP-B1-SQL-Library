# Financial Reports Queries

## Purpose
Comprehensive financial reporting queries including P&L, balance sheets, and financial analysis.

## Query Types
- Profit & Loss statements
- Balance sheet reports
- Cash flow statements
- Financial ratio analysis
- Period comparison reports
- Budget vs actual analysis

## Example File Names
- `FIN_RPT_Profit_And_Loss.sql`
- `FIN_RPT_Balance_Sheet.sql`
- `FIN_RPT_Cash_Flow_Statement.sql`
- `FIN_RPT_Monthly_Financial_Summary.sql`
- `FIN_RPT_Budget_Variance_Analysis.sql`

## SAP B1 Tables
Common tables used:
- `OACT` - Chart of Accounts
- `JDT1` - Journal Entry Rows
- `OJDT` - Journal Entry Headers
- `OAFS` - Financial Statement Structure
- `OACP` - Accounting Periods
