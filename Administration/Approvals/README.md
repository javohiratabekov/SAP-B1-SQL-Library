# Approval Process

SAP B1: **Administration → Approval Procedures**

Workflow rules that require managerial authorization before a document can be posted.

## Key Tables
- `OATP` — Approval Template (rules definition)
- `OATC` — Approval Stages
- `OWDD` — Waiting Documents (pending approval queue)

## Naming Prefix
`ADM_APR_` — e.g., `ADM_APR_Pending_Documents.sql`, `ADM_APR_Approval_History.sql`
