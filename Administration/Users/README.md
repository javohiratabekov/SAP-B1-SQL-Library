# Users

SAP B1: **Administration → Users**

User account management, license type assignments, and access authorization.

## Key Tables
- `OUSR` — Users
- `OUTB` — Authorization objects
- `OLOG` — User activity log

## Naming Prefix
`ADM_USR_` — e.g., `ADM_USR_Active_Users.sql`, `ADM_USR_Login_History.sql`
