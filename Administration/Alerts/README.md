# Alerts Management

SAP B1: **Administration → Alerts Management**

Query-based or event-based notifications sent to users when business conditions are triggered.

## Key Tables
- `OAMD` — Alert Message Definition
- `OAMS` — Alert Messages Sent log

## Naming Prefix
`ADM_ALT_` — e.g., `ADM_ALT_Low_Stock_Alert.sql`, `ADM_ALT_Overdue_Invoices.sql`
