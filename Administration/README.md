# Administration Queries

## Purpose
SQL queries for system administration, user management, and SAP B1 configuration.

## Query Types
- User and authorization queries
- System configuration reports
- Audit trail queries
- Database statistics
- Document numbering series
- Company and branch information

## Example File Names
- `ADM_User_List_And_Permissions.sql`
- `ADM_Audit_Trail_Report.sql`
- `ADM_Document_Series_Usage.sql`
- `ADM_Company_Configuration.sql`
- `ADM_System_Info_Report.sql`

## SAP B1 Tables
Common tables used:
- `OUSR` - Users
- `OAUT` - Authorizations
- `OADM` - Company Administration
- `ONNM` - Document Numbering Series
- `NNM1` - Number Series
- `OADP` - Document Types
