# Sales Returns Queries

## Purpose
SQL queries for sales returns, credit memos, and return analysis.

## Query Types
- Return authorization tracking
- Credit memo analysis
- Return rate by product
- Customer return patterns
- Return reason analysis
- Return processing status

## Example File Names
- `SAL_RET_Open_Returns.sql`
- `SAL_RET_Credit_Memo_Report.sql`
- `SAL_RET_Return_Rate_By_Item.sql`
- `SAL_RET_Customer_Return_History.sql`
- `SAL_RET_Return_Reason_Analysis.sql`

## SAP B1 Tables
Common tables used:
- `ORIN` - AR Credit Memo Header
- `RIN1` - AR Credit Memo Rows
- `ORDN` - Return Header
- `RDN1` - Return Rows
