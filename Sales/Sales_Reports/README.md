# Sales Reports Queries

## Purpose
Comprehensive sales reporting, analysis, and KPI tracking queries.

## Query Types
- Sales performance dashboards
- Sales by customer/item/region
- Trend analysis
- Sales forecasting data
- Commission calculations
- Sales pipeline reports

## Example File Names
- `SAL_RPT_Monthly_Sales_Summary.sql`
- `SAL_RPT_Sales_By_Customer.sql`
- `SAL_RPT_Sales_By_Item_Group.sql`
- `SAL_RPT_Sales_Trend_Analysis.sql`
- `SAL_RPT_Top_Customers_Report.sql`

## SAP B1 Tables
Common tables used:
- `OINV` - AR Invoice Header
- `INV1` - AR Invoice Rows
- `ORDR` - Sales Order Header
- `OCRD` - Business Partners
- `OITM` - Items Master Data
