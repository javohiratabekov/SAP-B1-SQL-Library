# Sales Delivery Queries

## Purpose
SQL queries for delivery notes, shipment tracking, and delivery performance analysis.

## Query Types
- Delivery notes tracking
- Shipment status
- Pending deliveries
- Delivery performance metrics
- Customer delivery history
- On-time delivery analysis

## Example File Names
- `SAL_DEL_Open_Deliveries.sql`
- `SAL_DEL_Shipment_Tracking.sql`
- `SAL_DEL_Delivery_Performance.sql`
- `SAL_DEL_Pending_Shipments.sql`
- `SAL_DEL_Customer_Delivery_History.sql`

## SAP B1 Tables
Common tables used:
- `ODLN` - Delivery Header
- `DLN1` - Delivery Rows
- `ORDR` - Sales Order Header
- `OSHP` - Shipment Types
