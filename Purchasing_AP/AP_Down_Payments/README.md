# A/P Down Payment Invoices

SAP B1: **Purchasing A/P → A/P Down Payment Invoice**

Advance payments made to vendors before goods receipt or service delivery.

## Key Tables
- `ODPO` — A/P Down Payment Header
- `DPO1` — A/P Down Payment Lines
