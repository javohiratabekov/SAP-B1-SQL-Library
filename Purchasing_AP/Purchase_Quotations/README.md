# Purchase Quotations (RFQ)

SAP B1: **Purchasing A/P → Purchase Quotation**

Request for Quotation (RFQ) sent to vendors for price comparison before ordering.

## Key Tables
- `OPQT` — Purchase Quotation Header
- `PQT1` — Purchase Quotation Lines

## Naming Prefix
`PUR_PQT_` — e.g., `PUR_PQT_Vendor_Price_Comparison.sql`
