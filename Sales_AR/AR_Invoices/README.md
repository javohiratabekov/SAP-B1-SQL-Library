# Sales Invoices Queries

## Purpose
SQL queries for AR invoices, billing analysis, and invoice tracking.

## Query Types
- Invoice listing and status
- Revenue recognition
- Invoice aging
- Customer billing analysis
- Payment status tracking
- Invoice line item details

## Example File Names
- `SAL_INV_Open_Invoices.sql`
- `SAL_INV_Monthly_Revenue.sql`
- `SAL_INV_Customer_Invoicing_History.sql`
- `SAL_INV_Unpaid_Invoices.sql`
- `SAL_INV_Invoice_Details_By_Item.sql`

## SAP B1 Tables
Common tables used:
- `OINV` - AR Invoice Header
- `INV1` - AR Invoice Rows
- `ORDR` - Sales Order Header
- `ODLN` - Delivery Header
