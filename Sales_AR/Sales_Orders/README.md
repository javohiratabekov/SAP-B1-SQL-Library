# Sales Orders Queries

## Purpose
SQL queries for sales order management, tracking, analysis, and reporting.

## Query Types
- Open sales orders
- Order fulfillment status
- Order backlog analysis
- Customer order history
- Order value tracking
- Delivery performance

## Example File Names
- `SAL_ORD_Open_Sales_Orders.sql`
- `SAL_ORD_Backlog_Report.sql`
- `SAL_ORD_Customer_Order_History.sql`
- `SAL_ORD_Pending_Delivery.sql`
- `SAL_ORD_Daily_Bookings.sql`

## SAP B1 Tables
Common tables used:
- `ORDR` - Sales Order Header
- `RDR1` - Sales Order Rows
- `OQUT` - Sales Quotation Header
- `OCRD` - Business Partners (Customers)
