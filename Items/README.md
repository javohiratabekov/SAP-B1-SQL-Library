# Items Queries

## Purpose
SQL queries for item master data, product information, and item analysis.

## Query Types
- Item master list
- Item pricing analysis
- Product catalog queries
- Item attributes and properties
- Item group analysis
- Active/inactive items

## Example File Names
- `ITM_Item_Master_List.sql`
- `ITM_Item_Pricing_Report.sql`
- `ITM_Active_Items_By_Group.sql`
- `ITM_Item_Properties_Details.sql`
- `ITM_Inactive_Items_Report.sql`

## SAP B1 Tables
Common tables used:
- `OITM` - Items Master Data
- `OITB` - Item Groups
- `OITW` - Item Warehouse Info
- `ITM1` - Item Prices
- `OBTN` - Batch Numbers (if applicable)
- `OSRN` - Serial Numbers (if applicable)
