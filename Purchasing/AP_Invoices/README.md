# AP Invoices Queries

## Purpose
SQL queries for vendor invoices, invoice matching, and payables tracking.

## Query Types
- AP invoice listing
- Invoice matching (3-way match)
- Invoice approval status
- Vendor invoice history
- Invoice discrepancies
- Payment terms tracking

## Example File Names
- `PUR_API_Open_AP_Invoices.sql`
- `PUR_API_Three_Way_Match_Report.sql`
- `PUR_API_Invoice_Discrepancies.sql`
- `PUR_API_Vendor_Invoice_History.sql`
- `PUR_API_Early_Payment_Discounts.sql`

## SAP B1 Tables
Common tables used:
- `OPCH` - AP Invoice Header
- `PCH1` - AP Invoice Rows
- `OPDN` - Goods Receipt PO Header
- `OPOR` - Purchase Order Header
