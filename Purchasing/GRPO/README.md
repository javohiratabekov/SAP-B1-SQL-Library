# Goods Receipt PO (GRPO) Queries

## Purpose
SQL queries for goods receipt tracking, receiving reports, and inventory receipts.

## Query Types
- GRPO tracking and status
- Receipt variance analysis
- Pending receipts
- Quality inspection tracking
- Receiving performance
- GRPO to invoice matching

## Example File Names
- `PUR_GRPO_Daily_Receipts.sql`
- `PUR_GRPO_Pending_Quality_Check.sql`
- `PUR_GRPO_Receipt_Variance_Report.sql`
- `PUR_GRPO_PO_To_Receipt_Status.sql`
- `PUR_GRPO_Receiving_Performance.sql`

## SAP B1 Tables
Common tables used:
- `OPDN` - Goods Receipt PO Header
- `PDN1` - Goods Receipt PO Rows
- `OPOR` - Purchase Order Header
- `POR1` - Purchase Order Rows
