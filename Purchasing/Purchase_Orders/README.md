# Purchase Orders Queries

## Purpose
SQL queries for purchase order management, tracking, and vendor performance.

## Query Types
- Open purchase orders
- PO tracking and status
- Vendor order history
- PO approval workflow
- Purchase commitments
- Order fulfillment tracking

## Example File Names
- `PUR_PO_Open_Purchase_Orders.sql`
- `PUR_PO_Pending_Approval.sql`
- `PUR_PO_Vendor_Order_History.sql`
- `PUR_PO_Overdue_Deliveries.sql`
- `PUR_PO_Purchase_Commitments.sql`

## SAP B1 Tables
Common tables used:
- `OPOR` - Purchase Order Header
- `POR1` - Purchase Order Rows
- `OCRD` - Business Partners (Vendors)
- `OITM` - Items Master Data
