## Naming Convention

All SQL files follow this standardized naming format:

**Format:** `MODULE_Function_Description.sql`

### Module Prefixes:
- `FIN` - Finance
- `SAL` - Sales
- `PUR` - Purchasing
- `INV` - Inventory
- `PRD` - Production
- `MRP` - Material Requirements Planning
- `BP` - Business Partners
- `ITM` - Items
- `HR` - Human Resources
- `DASH` - Dashboard Queries
- `ADM` - Administration

### Examples:
- `FIN_GL_Account_Balance.sql`
- `SAL_Open_Sales_Orders.sql`
- `INV_Stock_By_Warehouse.sql`
- `PUR_GRPO_Pending_Items.sql`
- `BP_Customer_Credit_Status.sql`
- `DASH_Monthly_Sales_Summary.sql`

## Usage
1. Navigate to the appropriate module folder
2. Review the README in each folder for specific query types
3. Use queries as templates or reference for your SAP B1 implementation
