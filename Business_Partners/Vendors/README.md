# Vendor Queries

## Purpose
SQL queries for vendor master data, performance analysis, and supplier management.

## Query Types
- Vendor master list
- Vendor performance metrics
- Payment terms analysis
- Top vendors by spend
- Vendor contact information
- Inactive vendor reports

## Example File Names
- `BP_VEND_Vendor_Master_List.sql`
- `BP_VEND_Top_Vendors_By_Spend.sql`
- `BP_VEND_Vendor_Performance_Report.sql`
- `BP_VEND_Payment_Terms_Analysis.sql`
- `BP_VEND_Inactive_Vendors.sql`

## SAP B1 Tables
Common tables used:
- `OCRD` - Business Partners
- `CRD1` - Business Partner Addresses
- `OPCH` - AP Invoice Header
- `OPOR` - Purchase Order Header
- `OCPR` - Contact Persons
