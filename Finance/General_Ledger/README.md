# General Ledger Queries

## Purpose
SQL queries related to General Ledger transactions, account balances, journal entries, and financial postings.

## Query Types
- GL account balances
- Journal entry reports
- Account movements and transactions
- Period-end balances
- Trial balance queries
- Chart of accounts analysis

## Example File Names
- `FIN_GL_Account_Balance.sql`
- `FIN_GL_Journal_Entries_By_Period.sql`
- `FIN_GL_Trial_Balance.sql`
- `FIN_GL_Account_Movements.sql`
- `FIN_GL_Unbalanced_Entries.sql`

## SAP B1 Tables
Common tables used:
- `OACT` - Chart of Accounts
- `JDT1` - Journal Entry - Rows
- `OJDT` - Journal Entry - Headers
- `OACP` - Accounting Periods
