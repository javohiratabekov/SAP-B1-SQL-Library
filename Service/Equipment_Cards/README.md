# Equipment Cards

SAP B1: **Service → Equipment Cards**

Master records for customer-owned equipment and assets tracked for service history.

## Key Tables
- `OSER` — Equipment Card (linked to serial number)
- `OSRN` — Serial Numbers master

## Naming Prefix
`SRV_EQP_` — e.g., `SRV_EQP_Customer_Equipment_List.sql`
