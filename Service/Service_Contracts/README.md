# Service Contracts

SAP B1: **Service → Service Contracts**

Agreements with customers defining service coverage scope, response times, and pricing.

## Key Tables
- `OSCN` — Service Contract Header
- `SCN1` — Contract Item lines

## Naming Prefix
`SRV_CON_` — e.g., `SRV_CON_Expiring_Contracts.sql`, `SRV_CON_Active_By_Customer.sql`
