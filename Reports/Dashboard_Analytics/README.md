# Dashboard Queries

## Purpose
High-level KPI and dashboard queries for executive reporting and business intelligence.

## Query Types
- Executive dashboard metrics
- Cross-functional KPIs
- Real-time business indicators
- Summary statistics
- Performance scorecards
- Trend dashboards

## Example File Names
- `DASH_Executive_Summary.sql`
- `DASH_Daily_Operations_KPIs.sql`
- `DASH_Sales_Purchasing_Overview.sql`
- `DASH_Financial_Snapshot.sql`
- `DASH_Inventory_Performance_Metrics.sql`

## SAP B1 Tables
Common tables used:
- Multiple tables across modules
- Aggregated data from:
  - Finance (OJDT, JDT1, OACT)
  - Sales (OINV, ORDR)
  - Purchasing (OPOR, OPCH)
  - Inventory (OITM, OITW, OINM)
