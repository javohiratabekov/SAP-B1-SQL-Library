# Checks

SAP B1: **Banking → Checks**

Manages printed outgoing checks and received customer checks.

## Key Tables
- `OCHQ` — Checks master
- `ORCT` — Linked to incoming payments (received checks)
- `OVPM` — Linked to outgoing payments (printed checks)

## Naming Prefix
`BNK_CHQ_` — e.g., `BNK_CHQ_Outstanding_Checks.sql`
