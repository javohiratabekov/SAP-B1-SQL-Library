# Bank Reconciliation

SAP B1: **Banking → Bank Statements and Reconciliation**

Matches SAP B1 payment transactions against external bank statement lines.

## Key Tables
- `OBNK` — Bank Statement Header
- `BNK1` — Bank Statement Lines
- `OBTF` — Bank Transfer records

## Naming Prefix
`BNK_REC_` — e.g., `BNK_REC_Unreconciled_Transactions.sql`
